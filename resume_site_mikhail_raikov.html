Проект / Arduino

GSM-сигнализация на Arduino Nano

Проект представляет собой охранную систему, которая фиксирует движение через инфракрасный датчик и отправляет SMS-уведомление на заданный номер через GSM-модуль. В рамках проекта выполнены сборка, пайка, подключение компонентов, написание логики работы и отладка.

  • Практика работы с микроконтроллерами и периферийными модулями.
  • Понимание связки «датчик → контроллер → канал уведомления».
  • Опыт диагностики и доведения физической системы до стабильной работы.

Проект / C++

Приложение для визуализации ориентированных графов

Учебное графическое приложение на C++ / MFC позволяет работать с взвешенными ориентированными графами: редактировать матрицу смежности, видеть вершины и рёбра, а также сохранять и восстанавливать данные через JSON.

  • Интерфейсная логика и работа с пользовательским вводом.
  • Связь между структурой данных и визуальным представлением.
  • Сохранение состояния приложения и загрузка ранее созданных данных.
